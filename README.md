# 🎵 Stray Kids — Fan Page

Sitio web informativo sobre el grupo surcoreano Stray Kids, con información de sus integrantes, discografía y contenido adicional de interés para fans.

🔗 **Demo en vivo:** https://pattyrivas.github.io/StrayKids/

## ✨ Funcionalidades

- Páginas con información de cada integrante
- Sección de discografía
- Navegación entre múltiples páginas
- Estilos organizados con SCSS (preprocesador CSS)

## 🛠️ Stack Tecnológico

- **HTML5** — estructura semántica multi-página
- **SCSS (Sass)** — preprocesador CSS para estilos modulares y mantenibles

## 🚀 Cómo ejecutar el proyecto localmente

```bash
# Clonar el repositorio
git clone https://github.com/Pattyrivas/StrayKids.git

# Entrar a la carpeta
cd StrayKids

# Abrir index.html en el navegador
# (o usar la extensión Live Server de VS Code)
```

Si modificas los archivos `.scss`, necesitas compilarlos a CSS (por ejemplo, con la extensión "Live Sass Compiler" de VS Code o el compilador de Sass vía npm).

## 📂 Estructura del proyecto
StrayKids/

├── css/         # Estilos compilados

├── sass/        # Archivos fuente SCSS

├── pages/       # Páginas adicionales del sitio

├── src/         # Recursos (imágenes, etc.)

└── index.html

## 📌 Aprendizajes aplicados

Proyecto enfocado en maquetación multi-página con HTML semántico y organización de estilos mediante SCSS: variables, anidamiento y modularización de archivos — una base más escalable que CSS plano.
